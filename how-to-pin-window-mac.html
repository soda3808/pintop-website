What macOS offers (and what it does not)

  • Split View / Stage Manager — layout helpers, not “always on top.”
  • Picture in Picture — video only, not docs or terminals.
  • Mission Control — navigation, not pinning.

There is still no built-in “pin this window above everything” for arbitrary apps.

Do it with PinTop (about 2 minutes)

1. Get the app
Download from Gumroad, open the DMG, drag PinTop to Applications.
2. Grant permissions
First launch: allow Accessibility and Screen Recording when prompted (System Settings → Privacy & Security).
3. Pin
Focus the window you care about. Press Ctrl+Opt+P. A live overlay stays on top. Press again to unpin.
4. Trial → license
Use full features for 7 days in-app. When it sticks, paste your Gumroad key. Activation is one-time; daily use stays offline.

Tips

  • Opacity shortcuts when you need to see through the overlay
  • Click-through when the overlay should not steal clicks
