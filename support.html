Frequently asked questions and troubleshooting for PinTop v2.0

Getting Started

How do I install PinTop?

Download PinTop from our Gumroad page. Open the DMG file and drag PinTop to your Applications folder. Double-click to launch. PinTop runs in your menu bar — look for the pin icon (📌) at the top of your screen.

macOS says "PinTop can't be opened because Apple cannot check it for malicious software"

This is a standard macOS Gatekeeper message for apps downloaded outside the Mac App Store. PinTop is safe — it contains zero network calls and collects no data. To open PinTop:

  1. Right-click (or Control-click) the PinTop app in your Applications folder
  2. Select "Open" from the context menu
  3. Click "Open" in the dialog that appears

You only need to do this once. After the first launch, macOS will remember your choice and PinTop will open normally.

What permissions does PinTop need?

PinTop requires two macOS permissions to function:

  1. Accessibility — Needed for global keyboard shortcuts and window snapping. Go to System Settings > Privacy & Security > Accessibility and enable PinTop.
  2. Screen Recording — Needed to capture window content for the overlay. Go to System Settings > Privacy & Security > Screen Recording and enable PinTop.

PinTop will prompt you for these permissions on first launch. You may need to restart PinTop after granting permissions.

How do I pin a window?

Click on the window you want to pin so it's in front, then press Ctrl+Option+P. The window will be captured as a floating overlay with a colored border. To unpin, press Ctrl+Option+P again.

Troubleshooting

PinTop says "Screen Recording permission required" but I already granted it

macOS sometimes requires a restart of the app after granting Screen Recording permission. Try these steps:

  1. Quit PinTop completely (right-click the menu bar icon > Quit)
  2. Go to System Settings > Privacy & Security > Screen Recording
  3. Toggle PinTop off, then back on
  4. Relaunch PinTop

If the issue persists, try restarting your Mac. macOS occasionally caches permission states.

Keyboard shortcuts don't work

This is usually an Accessibility permission issue. Check that PinTop is enabled in System Settings > Privacy & Security > Accessibility. If it's already enabled, try toggling it off and on, then relaunch PinTop.

Note: If another app uses the same shortcut (Ctrl+Option+P), there may be a conflict. Check other utility apps for overlapping shortcuts.

The overlay looks blurry or low resolution

PinTop captures at your display's native resolution. If the overlay appears blurry, try unpinning and re-pinning the window. This forces PinTop to recapture at the current display resolution, which helps when switching between monitors with different pixel densities.

PinTop uses too much CPU / battery

PinTop automatically reduces the capture frame rate from 30fps to 15fps when your Mac is on battery power. If you're still experiencing high CPU usage, try unpinning windows you're not actively using. Each pinned window uses a separate capture stream.

The pinned overlay doesn't follow the window when I move it

The overlay updates its position every 100ms. There may be a very brief delay when you move the original window. This is normal and the overlay will catch up almost immediately.
