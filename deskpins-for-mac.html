Why there's no DeskPins on macOS

On Windows, DeskPins lets you "pin" any window so it stays above every other window — a tiny tool thousands of people rely on daily. macOS has never offered this: there is no built-in always-on-top option, and DeskPins itself is built on Windows APIs that don't exist on the Mac.

PinTop was built to fill exactly that gap, natively, using Apple's ScreenCaptureKit. Pin any window with ⌃⌥P and it floats above everything — across Spaces and even over full-screen apps — while staying fully click-through, so you keep working in the window underneath.

Frequently asked questions

Is there a version of DeskPins for Mac?

No — DeskPins is Windows-only and has never been ported. PinTop is a native macOS app that covers the same core need: keeping any window on top of everything else.

How do I keep a window always on top on a Mac?

Install PinTop, grant it Accessibility and Screen Recording permissions, then press ⌃⌥P on any window. A colored border confirms the pin; press ⌃⌥P again to release it.

Does it work on Apple Silicon?

Yes — PinTop is a universal binary, native on both Apple Silicon and Intel, requiring macOS 14.0 (Sonoma) or later. The app is notarized by Apple.
