Always on Top for Mac

macOS does not ship a built-in “always on top” switch for arbitrary apps. PinTop adds it: pin docs, Figma, terminals, or a browser pane as a live floating overlay while you keep working underneath.

What “always on top” means on a Mac

On Windows, tools like DeskPins made it trivial. On Mac, people usually alt-tab, split view, or buy a full window manager. If you only want one window to float, that is overkill.

PinTop is a small menu bar app that pins any window as a live picture-in-picture style overlay — not a static screenshot.

How PinTop works

  • Global shortcut pins the frontmost window
  • Live capture so the overlay updates in real time
  • Scroll to resize, drag to move, opacity when you need to see through it
  • Click-through so the overlay does not steal your clicks
  • Mini corner PiP mode for dense screens

Who it is for

Developers following a build log. Designers mirroring a reference. Students keeping a PDF beside notes. Anyone tired of alt-tabbing the same two windows all day.
